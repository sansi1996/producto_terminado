# Sistema de Gestión Producto Terminado

Sistema web desarrollado para el sistema producto terminado en el Instituto Tecnológico Industrial Brasil Bolivia.

## Tecnologías usadas
- **Backend:** PHP
- **Base de datos:** MySQL
- **Contenedores:** Docker
- **Control de versiones:** Git / GitHub

## Módulos incluidos
- Productos y categorías
- Clientes y proveedores  
- Ventas y órdenes
- Pagos y devoluciones
- Salidas e insumos
- Auditoría y alertas
- Dashboard de control

## Archivos del proyecto
- Archivos `.php`: Código fuente del sistema
- `producto_terminado.sql`: Base de datos completa con estructura y datos
